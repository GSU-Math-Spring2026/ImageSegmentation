# Here are instructions to run the example

## 1. Open JupyterLab as intructed.
## 2. Remove the whole folder "ImageSegmentation", if you created one.
## 3. From File menu, choose New Launcher
## 4. Launcher, open Terminal
## 5. In Terminal, run this command: git clone https://github.com/GSU-Math-Spring2026/ImageSegmentation.git
## 6. Now you have a brand new folder ImageSegmentation. Browse and open the notebook inside.
## 7. Before running anything: from top right corner, switch kernel to Deep Learning (must be in the list of dialog that pops up):
